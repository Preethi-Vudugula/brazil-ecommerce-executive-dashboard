# brazil-ecommerce-executive-dashboard
Interactive Power BI dashboard analyzing sales, revenue, customer, and product performance using the Brazil E-Commerce (Olist) dataset.
Brazil E-Commerce Executive Sales Dashboard
Project Overview

This Power BI dashboard analyzes business performance using the Brazil E-Commerce Public Dataset (Olist). The dashboard provides executive-level insights into revenue, orders, customers, products, and payment behavior.
