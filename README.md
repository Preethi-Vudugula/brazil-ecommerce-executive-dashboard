Brazil E-Commerce Executive Dashboard
Project Overview

This project presents an executive-level business intelligence dashboard developed using Power BI, PostgreSQL, SQL, and DAX. The dashboard analyzes the Brazilian E-Commerce Public Dataset (Olist) to provide insights into revenue, customer behavior, product performance, and payment trends.
